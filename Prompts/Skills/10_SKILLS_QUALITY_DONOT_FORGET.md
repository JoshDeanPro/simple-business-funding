# Prompt: Subagent Skills Checklist

Use this checklist to verify subagent performance on codebase inspections.

---

## "Don't Forget" Subagent Skill Checklist

*   [ ] **SEO & Metadata**:
    *   Does every route contain page descriptions under 155 characters?
    *   Are JSON-LD schema objects valid?
*   [ ] **Responsive Quality**:
    *   Do email addresses wrap correctly?
    *   Do all select menus display correctly on mobile?
*   [ ] **Validation States**:
    *   Are form inputs clearly outlined on error?
    *   Does the submit button disable during submissions?
