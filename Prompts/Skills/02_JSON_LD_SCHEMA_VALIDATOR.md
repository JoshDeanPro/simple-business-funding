# Skill Prompt: JSON-LD Schema Auditor

Use this prompt to direct an AI specialist to check JSON-LD SEO schema configurations.

---

## Instructions
1. Verify the presence of standard schemas:
   - **Homepage**: `Organization` and `WebSite` schemas.
   - **Subpages**: `BreadcrumbList` schema.
2. Confirm that all URL keys are absolute.
3. Ensure no trailing commas exist in JSON objects.
