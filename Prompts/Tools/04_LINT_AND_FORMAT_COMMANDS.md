# Prompt: Linter and Prettier Command Guide

Use this prompt to run format check operations.

---

## Terminal Commands & Instructions
1. Run Prettier layout verification checks:
   ```bash
   npx prettier --check .
   ```
2. Automatically format files before committing:
   ```bash
   npx prettier --write .
   ```
3. Audit for code syntax and styling issues:
   ```bash
   npm run lint
   ```
