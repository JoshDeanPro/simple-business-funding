# Prompt: Asymmetrical UI Crop Shapes Guide

Use this prompt to audit and construct visual masks on imagery.

---

## Instructions
1. Avoid basic rectangular frames for major layout photographs.
2. Build custom double-curved geometric crop masks using CSS values (e.g., `rounded-[2.5rem] rounded-tr-[5rem] rounded-bl-[5rem]`).
3. Ensure that image masks align with text positions, creating a balanced page layout.
