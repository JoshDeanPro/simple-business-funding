# Prompt: Typography Scale and Case Directives

Use this prompt to establish clear reading sizes and casing formats.

---

## Instructions
1. Headings must use sentence case (e.g., "All library resources", "Frequently asked questions"). Avoid uppercase tracking.
2. Restrict typography sizes:
   - **Display H1**: `text-4xl sm:text-5xl lg:text-6xl font-extrabold tracking-tight`.
   - **H2 Page Section**: `text-2xl sm:text-3xl font-extrabold`.
   - **H3 Supporting Title**: `text-lg sm:text-xl font-bold`.
   - **Body Copy**: `text-sm sm:text-base leading-relaxed`.
   - **Legal Notices**: `text-xs leading-normal text-muted-text`.
