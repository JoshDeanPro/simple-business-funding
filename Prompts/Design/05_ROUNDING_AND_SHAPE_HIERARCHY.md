# Prompt: Rounding and Shape Hierarchy Specifications

Use this prompt to align border radii choices to visual elements.

---

## Instructions
1. Maintain the following shape hierarchy:
   - **Primary Action Elements**: Pill-shaped capsules (`rounded-full`) for high-contrast actions.
   - **Interactive Selectors & Chips**: Soft rounded corners (`rounded-xl` / 12px) for outcomes.
   - **Form Fields & Inputs**: Comfortable rounding (`rounded-lg` / 8px) for input borders.
   - **Support Panels**: Smooth rounded borders (`rounded-2xl` / 16px).
   - **Section Bands**: Square edges (`rounded-none`).
2. Never nest rounded panels with the exact same radius.
