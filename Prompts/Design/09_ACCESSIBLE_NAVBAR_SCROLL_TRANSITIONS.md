# Prompt: Dynamic Transparent-to-Solid Header Setup

Use this prompt to construct a scroll-responsive website header layout.

---

## Instructions
1. Monitor page scroll position using a React scroll listener.
2. Apply class transitions dynamically:
   - **At top (unscrolled)**: transparent background, white text and logo colors.
   - **Scrolled (>20px)**: solid white blurred background, dark text and cobalt blue logo.
3. Verify that the header remains readable against light backgrounds on subpages.
