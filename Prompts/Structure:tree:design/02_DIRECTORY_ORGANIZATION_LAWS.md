# Prompt: Directory Organization Principles

Use this prompt to enforce clean file separation rules.

---

## Instructions
1. Align file layout to these directories:
   - `src/routes/`: Route declarations.
   - `src/components/`: Shared site layouts.
   - `src/components/ui/`: Modular styling elements.
   - `src/styles.css`: CSS configurations and variables.
2. Do not leave temporary scratch files inside src directory folders.
