# Prompt: Architectural Code Structure Checklist

Use this checklist to ensure files are organized correctly and remain clean.

---

## "Don't Forget" Code Structure Checklist

*   [ ] **Folder Hierarchy**:
    *   Do all page layout components reside under `src/routes/`?
    *   Are generic helpers placed in `src/lib/`?
*   [ ] **Code Hygiene**:
    *   Have all console log lines been removed?
    *   Are CSS class lists ordered consistently?
*   [ ] **Imports**:
    *   Are absolute root shortcuts `@/` used instead of relative paths?
    *   Is the build output free of circular dependencies?
